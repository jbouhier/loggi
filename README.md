## Requirements

- Any Linux / POSIX system

## Install

- `pnpm i`

## Commands

- `pnpm dev` - local development
- `pnpm build` - Build for production
- `pnpm start` - Start production server
- `pnpm lint` - Lint checker
- `pnpm lint:fix` - Lint fix
- `pnpm types` - Check TS types
